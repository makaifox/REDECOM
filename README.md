# REDECOM App

Projeto criado em ReactJS com Typescript.


## *COMO INSTALAR ##

Na pasta raiz, rode um :

### `yarn install`

Então, rode :

### `yarn start`

para iniciar.


### `yarn test`


## *SOBRE O SISTEMA ##

o sistema esta hospedado na VPS da Heroku 

pode ser acessado pelo link ### `https://redecom.herokuapp.com/`

utilizando o bnco realtime do Firebase

**os acessos de admin de ambas as plataformas, são os mesmos do email de desenvolvimento**

## *ATUALIZANDO A HOSPEDAGEM ##

a VPS esta vinculada a branch master, a atualização acontece automaticamente após receber um push

**todas as alterações devem ser feitas primeiramente nas branchs de desenvolvimento, subindo apenas para a master após confirmação de sua funcionalidade**
